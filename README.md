# Shorts-generator-ai

An AI powered app for generating and editing short videos.

## What this project is about
Building an app that uses AI to help creators make short videos faster. 
Think: auto captions, AI clips, trending sounds, and quick edits.

## Goal
- Generate video ideas with AI
- Auto edit clips for TikTok, Reels, Shorts
- Learn coding + AI tools along the way
